# AdventureWorks Power BI Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using the AdventureWorks dataset.  
The goal is to analyze business performance by tracking sales, profit, production cost, customers, regions, and product trends to support data driven decision making.

## Objectives
- Analyze overall business performance
- Track sales, profit, and cost trends over time
- Compare sales and profit across regions
- Identify top performing customers and products
- Build an interactive and visually appealing dashboard

## Dashboard Features
### Key Metrics
- Total Customers
- Total Sales Amount
- Total Profit
- Profit Margin
- Production Cost

### Business Insights
- Monthly profit trend analysis
- Sales vs profit comparison by region
- Sales vs production cost by year
- Profit distribution by year
- Profit analysis by quarter
- Top customers by sales and profit
- Product wise sales performance

### Interactivity
- Month slicer
- Year slicer
- Region based filtering
- Interactive map visualization
- Dynamic cross filtering across visuals

## Tools and Technologies
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization

## Dataset
- AdventureWorks sample dataset
- Includes sales, customers, products, geography, and time based data
